\begin{Lem}
Let $X$ be a local martingale. Then $X$ is a continuous martingale iff the set $\{X_{\tau} : \tau \ \text{stopping \ time}, \tau \leq c\}$ is uniformly integrable for each $c \geq 0$.
\end{Lem}
