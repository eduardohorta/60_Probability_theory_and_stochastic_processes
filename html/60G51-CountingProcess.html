counting process


A stochastic process {X⁢(t)∣t∈ℝ+∪{0}} is called a counting process if, for each outcome ω in the sample space Ω,

  1. 1.

    X⁢(t)∈ℤ+∪{0} for all t,

  2. 2.

    X⁢(t)⁢(ω) is piecewise constant,

  3. 3.

    X⁢(t)⁢(ω) is non-decreasing,

  4. 4.

    X⁢(t)⁢(ω) is right continuous (continuous from the right), and

  5. 5.

    for any t, there is an s∈ℝ such that t<s and X⁢(t)⁢(ω)+1=X⁢(s)⁢(ω).

Remark. For any t, the random variable X⁢(t) is usually called the number of occurrences of some event by time t. Then, for s<t, X⁢(t)-X⁢(s) is the number of occurrences in the half-open interval (s,t].
