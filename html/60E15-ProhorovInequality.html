Prohorov inequality


Theorem (Prohorov inequality, 1959):

Let {Xi}i=1n be a collection of independent random variables satisfying the conditions:

a) E⁢[Xi2]<∞ ∀i, so that one can write ∑i=1nE⁢[Xi2]=v2
b) Pr⁡{|Xi|≤M}=1  ∀i.

Then, for any ε≥0,

Pr⁡{∑i=1n(Xi-E⁢[Xi])>ε} ≤ exp⁡[-ε2⁢M⁢arsinh⁡(ε⁢M2⁢v2)]
Pr⁡{|∑i=1n(Xi-E⁢[Xi])|>ε} ≤ 2⁢exp⁡[-ε2⁢M⁢arsinh⁡(ε⁢M2⁢v2)]

(See here (http://planetmath.org/AreaFunctions) for the meaning of arsinh⁡(x))
